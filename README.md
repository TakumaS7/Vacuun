# Vacuun!



## 概要

吸って吐き出せ！可愛い弾丸シューティングアクションゲーム！



## 技術スタック

- C、C++
- DirectX11
- XAudio2
- ImGui
- yaml-cpp
- Visual Studio 2022



## チーム開発での役割

主にコントローラーの開発に専念しました。
自分で制作した操作で、チームのみんなに遊んでもらい、レビューを沢山いただきながら改善を行っていました。
また、ゲームのUI、敵にも一部携わりました。



## 課題と解決

最初、コントローラーはXboxコントローラーを採用していました。
しかし、テストプレイにおいて操作に直感性がないことが課題になりました。
そこで、Wiiリモコンとヌンチャクを採用し、プレイヤーが吸う、吐くの動作を直感的に行えるようにしました。
これにより、ゲームへの没入感も増し、フィードバックも好評でした。



## 得られた学び

今作はチーム開発ということで、コミュニケーションの重要性を改めて学びました。
チームメンバーと協力することで、一人では成しえない作品を作ることが出来たと感じています。
また、コントローラーの開発を通じて、ユーザーエクスペリエンスの向上に対する意識も高まりました。



## 操作方法

コントローラーはWiiリモコンとヌンチャクを使用します。
接続方法は別途readmeフォルダにある、「Wiiリモコン接続方法」をご参照ください。

- Wiiリモコンを振る：敵を吸う
- ヌンチャクZボタン：吸った敵を吐き出す
- ヌンチャクスティック：移動
- ヌンチャクCボタン：ウルト発動
